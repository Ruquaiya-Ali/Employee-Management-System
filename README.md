# Employee Management System
A C++ OOP project built for ENGTECH 1PR3 at McMaster University.

## Features
- Add, remove, and search employees
- Three employee types: Employee, Manager, Intern
- Payroll report with salary calculations
- File I/O — data saves and loads automatically

## Concepts Used
- Inheritance, Polymorphism, Abstract Classes
- Virtual Functions, Encapsulation
- Dynamic Memory, File Handling, Static Members

## How to Run
Compile all .cpp files together and run the executable.
Place employees.txt in the same directory as the executable.
